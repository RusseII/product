# Bond config

- Maximum interest rate
  - Unit: %
  - Ex: 15%
- Term length
  - Unit: months
  - Ex: 24 months
- Collateral
  - Contract address
    - Ex: 0x1f9840a85d5af5bf1d1762f925bdaddc4201f984 (can be shortened)
  - Token image URL
  - Amount of tokens
    - Ex: 20,000
- Convertibility
  - Yes/No
